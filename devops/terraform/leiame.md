# Começando
1. instale o azure cli 
2. instale o kubectl
3. logar no azure cli
   az login 
   ## lista todos as zonas de resource groups 
   $ az account list-locations -o table
   $ az vm list-sizes --location -o table

3.criar arquivo main.tf
4. terraform init
5. terraform plan
6. terraform apply
## Aplicada o terraform  
7. terraform apply --auto-approve
